About

SpaceRDF is about bringing together Hackerspaces and the Semantic Web. Using Semantic Web technologies, SpaceRDF aims to provide a uniform way of publishing data about hackerspaces.

The inital aim is to create an OWL ontology, extending other ontologies existing ontologies where appropriate, that allows for the representation of all the data that can be represented using SpaceAPI. Initially, no effort will be made to seperate the physical space from the organisation, although this may be considered later. SpaceAPI is designed to show a standalone snapshot of current data whereas the SpaceRDF schema will be designed to allow for the archiving of data and linking with other datasets. It should be noted that SpaceRDF is not intended to replace SpaceAPI, they both have different aims and will complement each other.

Once an initial OWL ontology has been created, a document describing a mapping between SpaceAPI terms and SpaceRDF terms will be written. The second aim of this project is to create an archive, that can be queried using SPARQL, of data published by hackerspaces using SpaceAPI.
